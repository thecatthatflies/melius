# melius
> melius: latin for "better"

## overview
- melius is a simple, numbed down and end-to-end customizable code editor, built on [monaco](https://microsoft.github.io/monaco-editor/) and [electron](https://www.electronjs.org/).
- like vscode, it's open source, has extension support, and includes a file browser and terminal. melius also has other customization features such as fined tuned themes, custom application fonts, prebuilt code templates, and more.
